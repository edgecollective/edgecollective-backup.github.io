Solarpunk Infrastructure for Great Good!

Above: Vincent Callebaut, Asian Cairns, Sustainable Megaliths for Rural Urbanities, Shenzhen China

Overview

Climate, food, water, energy, communications -- let's learn to build our own resilient infrastructure for all of these challenges!

The following series of classes ("SIGG: Solarpunk Infrastructure for Great Good") represents a 'crash course' in DIY electronics applications, with an emphasis on:

  • collaborative, open design
  • decentralized production
  • off-grid capability, and
  • flexible supply chain options

Applications of this course material include:

  • water monitoring (floods, drought)
  • solar power generation and monitoring
  • maintaining off-grid farms
  • controlling lighting and motors off-grid

Class structure

Each class is heavily project-based; the aim of the course is to produce practical, actionable knowledge in a peer-learning setting.

Prequisites

  • Basic Electronics, or equivalent (or permission of instructor)
  • Some familiarity with Python or C programming

Schedule

This course series includes a series of classes -- "SIGG 1, SIGG 2, SIGG 3" etc; each class (e.g. "SIGG 1" consists of four meetings); each meeting is held for two hours, once per week. The classes build upon one another, but it is possible for students to register for classes individually.

References

  • A. Flynn, "Solarpunk: Notes Towards a Manifesto"

  • B. Valentine and V. Callebaut, "Solarpunk wants to save the world"
